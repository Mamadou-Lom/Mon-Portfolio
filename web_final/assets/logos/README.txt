=============================================
  LOGOS DES COMPÉTENCES — MODE D'EMPLOI
=============================================

Télécharge le logo de chaque outil et place-le ici
avec le nom de fichier EXACT indiqué ci-dessous.

Formats acceptés : PNG, JPG, SVG, WEBP
(change l'extension dans index.html si besoin)

Noms de fichiers attendus :
----------------------------------------------
python.png       → logo Python
r.png            → logo R / RStudio
sql.png          → logo SQL (MySQL, PostgreSQL…)
sas.png          → logo SAS
vba.png          → logo VBA (ou Excel avec "VBA")
excel.png        → logo Microsoft Excel
powerpivot.png   → logo PowerPivot
powerbi.png      → logo Microsoft Power BI
qliksense.png    → logo Qlik Sense
latex.png        → logo LaTeX
markdown.png     → logo Markdown
vscode.png       → logo Visual Studio Code
jupyter.png      → logo Jupyter
github.png       → logo GitHub

Où trouver les logos officiels :
----------------------------------------------
• https://seeklogo.com
• https://worldvectorlogo.com
• https://brandfetch.com
• Google Images > "logo XXX PNG transparent"

Conseil : préfère des images PNG avec fond transparent
et d'au moins 200x200px pour une bonne qualité.
